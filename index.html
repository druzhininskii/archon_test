doc→md

        
DOCX to Markdown Converter

      

    

    

      
This tool converts Microsoft Word documents (.docx) into clean, portable Markdown text — directly in your browser.

      
    
        
  • Drag & drop or browse — select any .docx file from your computer.
    • 
        
  • One-click conversion — the file is sent to the server, parsed, and returned as Markdown instantly.
    • 
        
  • Source & Preview tabs — inspect the raw Markdown or see a rendered HTML preview side by side.
    • 
        
  • Copy or Download — grab the Markdown to your clipboard or save it as a .md file.
    • 
        
  • AI Summarization — generate a short, medium, or detailed summary of the document using a language model of your choice.
    • 
      

      
All conversion happens server-side via FastAPI + mammoth. No files are stored — your document is processed in memory and discarded immediately.
